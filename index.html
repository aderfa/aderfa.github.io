VXFAMS — Anti-Cheat & Server PVP

Fair play itu bukan slogan, tapi kerjaan sehari-hari

Sehari-hari saya megang VXFAMS, server FiveM dengan mode PVP kayak TDM dan Battle Royale. Bagian yang paling banyak makan waktu justru bukan bikin fitur baru, tapi mikirin gimana caranya cheater nggak bisa lewat — mulai dari deteksi silent aim di sisi server, validasi damage per tembakan, sampai nge-lock proses game biar nggak gampang di-inject atau di-alt-tab pas lagi match.

Lua (FXServer) Anti-Silent-Aim WaveShield Integration C++ Process Monitor Matchmaking System

SEKARANG

Anti-Cheat & Server Developer — VXFAMS

Fokus keamanan game & arsitektur server PVP

  • Bikin sistem anti-silent-aim mulai dari validasi damage sisi shooter sampai grace period buat gerakan flick shot yang wajar.
  • Nyambungin deteksi armor ilegal dan kill-spam ke WaveShield buat proses ban otomatis.
  • Bikin aplikasi C++ terpisah buat monitoring proses game — ngunci fokus window, ngebatasin cursor, sampai blokir Alt+Tab pas match lagi jalan.
  • Benerin dan ngembangin sistem matchmaking/lobby buat mode TDM, termasuk bot AI, map & weapon picker, dan alur pindah antar match.
SEBELUMNYA

FiveM Scripter & Reverse Engineering

Memory hooking & eksperimen sistem cheat-detection

  • Ngulik vtable hooking dan pattern scanning di GTA V buat ngerti gimana cheat kelas berat biasa bikin exploit.
  • Dari situ balik lagi ke sisi bertahan: gimana caranya server bisa nangkep pola-pola yang sama sebelum jadi masalah beneran.

Proyek yang Lagi/Pernah Digarap

Sebagian besar berputar di sekitar satu masalah: gimana bikin PVP tetap fair.

antiSA (Anti Silent-Aim)

Sistem deteksi silent aim di sisi server. Dari sekadar invincibility gating, berkembang jadi validasi per-tembakan di sisi shooter plus grace period biar flick shot yang wajar nggak ke-flag salah.

Lua FiveM WaveShield

Game Overlay & Process Monitor

Aplikasi Windows di C++ yang hook ke overlay Steam pakai ImGui + DirectX 11, dipasangkan sama tool monitoring proses yang ngunci fokus, ngebatasin cursor, dan nge-block Alt+Tab selama match FiveM berlangsung.

C++ ImGui Win32

vx_rank — Matchmaking & Lobby

Sistem matchmaking untuk mode TDM: lobby custom dengan slot tim, undangan pemain, map & weapon picker, plus integrasi ke sf_tdm buat ngatur bot AI dan alur pertandingan.

Lua NUI/JS MySQL
